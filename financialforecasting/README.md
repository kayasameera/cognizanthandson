# Financial Forecasting using Recursion

## Description

This project predicts the future value of an investment using a recursive algorithm.

## Formula

Future Value = Present Value × (1 + Growth Rate)^Years

## Features

- Recursive implementation
- Java
- Simple financial forecasting
- Easy to understand

## Example

Initial Amount = ₹10000

Growth Rate = 10%

Years = 5

Output

Future Value = ₹16105.10
